## Descrição
Jogo desenvolvido como tarefa prática para o curso **Python Pro**, utilizando interfaces gráficas e lógica funcional.

---

## Compatibilidade por Sistema

O **Tkinter** é a biblioteca padrão do Python, mas sua instalação varia conforme o sistema operacional:

| Sistema | Como Garantir o Funcionamento |
| :--- | :--- |
| **Windows** | Geralmente já incluso na instalação padrão do Python. |
| **macOS** | Incluso no Python, mas pode exigir a instalação do [Tcl/Tk](https://www.python.org/download/mac/tcltk/) em versões antigas. |
| **Linux** | Requer instalação manual do pacote gráfico via terminal. |
| **WSL (Linux no Windows)** | **Não recomendado.** Exige um Servidor X configurado para exibir janelas. Use o PowerShell do Windows. |
